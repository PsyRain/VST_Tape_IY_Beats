# Changelog — Tape IY Beats

Формат: [версия] — дата. Даты в формате ГГГГ-ММ-ДД.

## [1.0.0] — 2026-06-07

Первый публичный релиз.

**Звук**
- Музыкальный wow/flutter: кубическая Hermite-интерполяция, многослойная
  модуляция (wow + flutter + scrape + дрейф), реалистичные глубины.
- Ленточное насыщение (модель намагничивания + оверсэмплинг 2×, гарантия ≤ 0 dBFS).
- Ленточное эхо, синхронизированное к темпу хоста.
- Износ ленты: плавные выпадения (dropouts), reverse-глитчи, шум (hiss/гул/рокот),
  print-through / пре-эхо.
- Режимы машин: Кассета · Катушка · VHS · Диктофон (перенастраивают весь движок).
- Tape Stop — варифспид-тормоз с провалом высоты тона; при стопе эффекты глушатся.
- Тон (Тепло): LP + head bump.

**Интерфейс**
- 6 ручек, селектор режима, BYPASS, Tape Stop.
- Анимированный видео-фон (ping-pong) с кнопкой стоп/пуск (стоп = статичная картинка).
- Подсказки при наведении, кликабельная ссылка на сайт.

**Технически**
- VST3 + Standalone, Windows 10/11 (64-bit), JUCE 8.
- Имена параметров латиницей (корректное отображение в хостах).
- Сообщение латентности хосту (PDC).
